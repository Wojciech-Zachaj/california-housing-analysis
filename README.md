## 🎯 Objective
# Analysis of Real Estate Prices in California

## 🎯 Project Objective
The objective of the project was to examine the correlation between residents’ average income and the market value of real estate in California, and to build a predictive model.

## 📊 Key Findings
- A strong positive correlation (0.67) was identified between income and home prices.
- The visualization revealed the so-called “ceiling effect” for values above $500k, which represents a limitation of the dataset.
- The linear regression model achieved an R-squared value of 0.50, confirming the high accuracy of the selected independent variable.
